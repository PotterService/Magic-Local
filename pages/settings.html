Export Owned / Saved Cards

Exports cards where you entered owned, foil, condition, or notes.

Import Owned / Saved Cards

Import a backup you exported earlier.

Remove Cards Not Owned

Deletes downloaded card database records where owned and foil are both 0. This frees space but keeps cards you own.

Clear Downloaded Card Data

Deletes downloaded cards and set list from this browser. Your collection entries stay unless you clear those too.

Clear Collection Entries

Deletes owned counts, foils, condition, and notes.

Clear Everything

Deletes cards, sets, prices, and collection entries from this browser.

Set Cover Notes

Missing cover files no longer auto-request random local files, so the 404 errors should stop. To add custom covers, edit data/set-covers.json and point set codes to your image files.

{ "znr": "assets/set-covers/znr.jpg" }

Reminder

Storage cleanup can free up space. If you remove not-owned cards, cards marked as owned stay saved. You can still redownload those sets later.
