# Magic Local

Magic Local is a standalone Magic: The Gathering collection management application designed for collectors, players, traders, and deck builders.

The application allows users to manage their collection entirely on their own device without requiring cloud accounts or subscriptions.

Features include:

* Collection tracking
* Deck building
* Wishlist management
* Trade binder management
* Collection statistics
* Card search and filtering
* Set tracking
* Duplicate tracking
* Collection value tracking
* Local data storage
* Binder organization

Magic Local is built for users who want full control over their collection data while maintaining a fast, simple, and reliable experience.
